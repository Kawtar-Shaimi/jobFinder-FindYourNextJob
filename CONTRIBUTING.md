# Contributing to JobFinder

## Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run server` to start JSON Server on port 3000
4. Run `npm start` to start the Angular dev server

## Branch Strategy
- `main` - production-ready code
- `feature/*` - new features
- `fix/*` - bug fixes

## Commit Convention
Follow Conventional Commits: `feat|fix|style|refactor|docs|chore|test: message`

## Code Style
- Use standalone components (Angular 19 pattern)
- One component per file
- Keep components focused and small
- Write clear, readable code
